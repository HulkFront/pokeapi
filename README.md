🏁 Execute o projeto localmente:
# Clone este repositório
$ git clone https://github.com/HulkFront/pokeapi.git

# Acesse a pasta do projeto no terminal/cmd
$ cd pokeapi

# Instale as dependências
$ npm install

# Execute a aplicação
$ npm run dev
